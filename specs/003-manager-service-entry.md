# Specification 003 — Manager service entry

## Goal

Give a manager a working operational dashboard to record completed services and monitor the barber team's daily activity.

## Acceptance criteria

1. A manager can select an existing barber and record a positive service amount.
2. Unknown barbers and zero/negative/non-finite amounts are rejected without a service being saved.
3. The manager dashboard shows today's revenue, service count, active barbers, barber revenue, and recent services.
4. The manager interface does not display CEO-only expense or net-profit KPIs.
